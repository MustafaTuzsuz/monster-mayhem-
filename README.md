# Monster Mayhem

A browser-based board game played on a 10x10 grid of hexagons.
Built with vanilla HTML, CSS and JavaScript. No server-side code, no build step.

CCT College Dublin assessment submission.

## Running
Open index.html in a browser.

## Development process
This project was developed with AI assistance (Claude Code), used under a fixed
set of project rules defined in CLAUDE.md. Every generated iteration is committed
separately with the prompt recorded in the commit body. Manual modifications are
committed separately. Troubleshooting steps are documented in the commit messages,
including attempts that did not resolve the issue.
